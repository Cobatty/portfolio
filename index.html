Pig Game
HTML / CSS / JavaScript
Git Hub
JavaScriptの基本文法を学ぶため、オンラインのコース(Udemy.com)を受講しながら作成したWebアプリ（2人用対戦ゲーム）です。いち早く基準の得点を獲得したプレーヤーの勝利です。

ルール1: プレイヤーは自分の番に、ダイス1個を何度でも振ることができます。いつ振りやめても構いません。
ルール2: HOLDを押した時点で出目の合計がその番における得点として合計得点に加算され、相手の番に変わります。
ルール3: 1の目が出たら得点0で自分の番が終了します。6の目が2回連続で出ると、合計得点が0になります。
